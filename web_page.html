There are several ways to replace missing values, for example the missing values in a column can be replaced by its mean, its median or its own most frequent value, depending on the type of data in this column.

  • If the data type in this column is contininous (float), then its missing values should be replaced by its median ot its mean. Because the probability that a continous variable equals to a given value is 0, hence we can not replace the missing values in this case by the most frequent value of the column.
  • For the columns which consist of categorical variables, the missing values in these columns can be replaced by its own most frequent values.
  • For columns containing discrete numercical variables, we can use all three given methods to replace their misisng values (replace by their mean, their median or their most frequent value).

Our dataframe consists of two types of data: float and object. Hence, we replace all the missing values in the float columns by their means, and all missing values on the object columns by their most frequent values.

IV. PCA in detail¶

PCA method consists of following steps:

  • Step 1: Extract $X$ and $y$ from the dataframe
    • The values of $X$ are extracted from some columns which present the amount of nutrient in 100g,
    • $y$ contains the values of the column "nutrition-grade-fr"
  • Step 2: Data nomalization
  • Step 3: Determine the covariance matrix \begin{equation} \Sigma = \dfrac{1}{n-1}(X - \mu)^T(X-\mu). \end{equation} where $n$ is the total number of observations and $\mu$ is the mean.
  • Step 4: Compute the eigenvalues $\Lambda = \{\lambda_1, \dots, \lambda_p \}$ and eigenvectors $\Gamma = \{ v_1, \dots, v_p \} $ of $\Sigma$
  • Step 5: Select principal components $\{ \tilde{v}_1, \dots, \tilde{v}_k \}$, $(k < p)$. \
    The explained-variance $\sigma_i$ on $i^{th}$-component is defined by: \begin{equation} \sigma_i = \dfrac{\lambda_i}{\sum_{i = 1}^p \lambda_i}. \end{equation}

It tells us how much information which is attributed on the $i^{th}$-component. Hence, we aim to select the components at which the explained variances are the largest.

  • Step 6: Project data on the selected components. Let's define $W$ be the subspace of $\mathbb{R}^p$ which is generated by selected principal components: $W = < \tilde{v}_1, \dots, \tilde{v}_k >$. The projection of data on $W$ is determined by $X_{projected} = X * W$, $X_{projected} \in \mathbb{R}^{n \times k} (k < p)$.

VI. Test ANOVA¶

1. A simple outline of one-way ANOVA test¶

Given $K$ classes of data $\{Y_i^{(k)} \}_{i = \overline{1, n_k}, k = \overline{1,K}}$}, where $n_k$ is the observation number of $k^{th}$-class. Suppose that their means are unknown. ANOVA test is a method which involves in comparing the means of these classes. \ $H_0: \mu_1 = \mu_2 = \dots = \mu_K$. \ $H_1$: Not all $\mu_i$ are equal, where $i = 1, \dots, K$. \

The $F$-test formula is defined by: \begin{equation} F^* = \dfrac{ \frac{1}{k-1} \Sigma_b}{ \frac{1}{n_t - K} \Sigma_w} \sim F_{K-1, n_t - K}. \end{equation} where $\Sigma_w$ and $\Sigma_b$ are respectively the covariance within and between classes, given by: \begin{equation} \Sigma_w = \sum_{k = 1}^K \sum_{i = 1}^{n_k}(Y^{(k)}_i - \mu_k)^2 \text{ and } \Sigma_b = \sum_{k = 1}^K n_k (\mu_k - \overline{Y})^2 \end{equation}

For a given significance level $\alpha$:

  • Accept $H_0$ when $F^* < F_{\alpha; K-1, n-K}$ or $p_{\text{value}} > \alpha$,
  • Reject $H_0$ when $F^* > F_{\alpha; K-1, n-K}$ or $p_{\text{value}} < \alpha$.

Remark: The default effect size reported in Pingouin is the partial eta-square, given by: \begin{equation} \eta_p^2 = \dfrac{\Sigma_b}{\Sigma_{total}} = \dfrac{ \sum_{k = 1}^K n_k (\mu_k - \overline{Y})^2}{\sum_{ k = 1}^K \sum_{i = 1}^{n_k}(Y^{(k)}_i - \mu)^2}. \end{equation}

Return to our project, let's take the column "nutrition_grade_fr" as the response feature. This columns consists of five groups (a, b, c, d, e). Our purpose is to test whether the means all numerical features are equal between groups. There are several Python packages which support to this test, for example pingouin, scipy, statsmodel, pandas. In this project, we only focus on the first two packages Pingouin and Scipy.

VII. Conclusion¶

  • In this project, we have discovered a completed procedure of the data analysis, from the basic steps as data importing and cleaning to dimensional reduction as well as data testing.
  • Through these analysis, we extract from the data a lot of important information, including:
    • The distribution of nutrition amount over 100g of product.
    • The correlation between variables.
    • The difference of means between groups,
    • ...

  • The variance-covariance matrix is an important item in statistic. It is not only used for presenting the relationships between variables, it is also a premise for both PCA and ANOVA methods. The objective of PCA is finding the components at which the projection of data are most discriminated. This method is really helpful for dimensional reduction, by keeping only the most important features. This make a lot of sense in data data analysis, especially in high dimensional case, as:

    • Simplifying the data visualization,

    • Saving stockage, computational time,

    • Improving the performance of Machine learning methods, etc

  • Through analyzing the variances between groups, the ANOVA test import also the information of means between these groups. There are many Python packages which support for this test, for example pingouin, statsmodels, pandas, scipy.
